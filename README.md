## MongoDB Atlas

- MongoDB https://mongodb.com
- Allowed IP addresses can only be added through the MongoDB Atlas website
- MongoDB Compass: the desktop GUI client for MongoDB
- A MongoDB URI is required to connect to the database
- Be careful not to upload the MongoDB URI to GitHub

## Deployment to Render.com

https://render.com/ Launch new site

- Upload the project to be deployed to a GitHub repository
- Log in to Render → New Web Service → connect your GitHub account
- Add environment variables directly in Render
- Configure MongoDB Atlas security (add allowed IP addresses)
- The free plan allows up to 750 hours of usage per month
